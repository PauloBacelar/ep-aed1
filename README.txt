EP - Indexidador e buscador de palavras

Sistema de indexação e busca de palavras em um arquivo de texto.

Para compilar, certifique-se de ter o GCC (compilador C) instalado na sua máquina.

Ao abrir terminal/powershell, compile o seu programa executando o comando:
  gcc main.c -o main

Após compilar, execute:
  ./main <arquivo> lista
caso queira utilizar a lista como estrutura de dados
  ./main <arquivo> arvore
caso queira utilizar a árvore como estrutura de dados

Em seguida, para buscar uma palavra, digite:
  busca <palavra>

Para finalizar o programa, basta digitar:
  fim
